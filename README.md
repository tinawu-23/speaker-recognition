# Speaker Recognition
## Usage
To run model1:     
`cd model/src`    
`python predict.py` 
To run model2:     
`cd model/src2`    
`python predict.py` 

To run application:     
`python app.py`  
Make sure your uploaded audio files are called key.wav, key1.wav and user1.wav, user2.wav, and user3.wav or change model/meta/sets.txt

## Requirements
`pip install -r requirements.txt`
- Python 3.6.5    
- Tensorflow 1.4.0    
- Keras 2.1.4
